🏗️ 建築施工工期估算輔助系統 (Construction Schedule Estimator)
這是一個基於 Python 與 Streamlit 開發的專業營建工期估算工具。專為建築師、專案經理與工程估價人員設計，能夠根據建築規模、結構型式、地質條件及施工工法，快速生成工期預估表與甘特圖。

目前版本：v6.17

✨ 主要功能特點
1. 精細的參數設定
結構分流：可獨立設定「地下結構」(RC/SRC) 與「地上結構」(RC/SRC/SS/SC)，反映真實施工速率差異。

基礎與開挖：

支援多種基礎型式：含 壁樁 (Barrette)、全套管基樁、微型樁等。

開挖工法邏輯判定：自動區分「放坡開挖」(無支撐) 與「擋土支撐」工期。

連續壁輔助工法：可複選地中壁、扶壁，自動加成工期。

工程範圍彈性：可勾選是否納入 機電、裝修、景觀 工程，適用於結構標或毛胚交屋案。

2. 智慧排程運算 (CPM)
順打 vs 逆打：自動判斷工法，調整地下室結構與地上層結構的併行/序列關係。

工項拆解：將「擋土壁施作」、「擋土支撐架設」、「土方開挖」與「地下結構」完全獨立拆分，符合工地實務。

規模連動：依據基地面積、樓地板面積自動計算工期係數 (Scale Factor)。

3. 視覺化報表
專業甘特圖 (Gantt Chart)：使用 Plotly 繪製，具備最佳化的視覺參數 (色塊高度 0.5、字體 16px)，閱讀清晰。

詳細進度表：自動過濾工期為 0 的項目，顯示清晰的起訖日期。

Excel 導出：一鍵下載 .xlsx 報表，格式整齊，包含所有參數設定與進度細節。

🚀 快速開始
環境需求
請確保您的環境已安裝 Python 3.8+，並安裝以下套件：

Bash

pip install streamlit pandas plotly openpyxl
執行程式
在終端機 (Terminal) 執行以下指令啟動系統：

Bash

streamlit run app.py
🧮 核心運算邏輯說明
地下結構：基準為 35天/層 (考量施工難度)，並加上基礎型式 (如全套管、壁樁) 的額外天數。

地上結構：依據結構材質 (SC/SS 較快，RC 較慢) 設定不同的單層工期。

開挖與支撐：

若選擇「放坡開挖」，支撐架設工期自動歸零。

若選擇「連續壁」，擋土工程時間會顯著增加。

屋突層 (PH)：屋突層數會併入地上結構工期計算，反映電梯機房等必要設施之施作時間。
